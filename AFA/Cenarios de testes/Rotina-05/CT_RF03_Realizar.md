## Cenário 03: Realizar retirar valores

### Caso de Teste 01: Retirar valores pelo sistema .

| ID       | Descrição                                                  |
| :------- | :-------------------------------------------------------   |
| C03-CT01 | retirar valores pelo sistema           |

| **Pré-condições**                                                     |
| :------------------------------------------------------------         |
| caixa deve estar aberto                                               |

| **Passos**                                                            |
| :----------------------------------------------------------------     |
| **DADO** que estou na tela de "Livro Caixa"                           |
| **QUANDO** clico em retirar valores                                   |
| **ENTÃO** o sistema exibe a tela para retirar valor                   |
| **E** preencho o campo com o valor de retirada                        |
| **QUANDO** clico em "Salvar"                                          |
| **ENTÃO** o valor e retirado                                          |

| **Critérios de aceitação**                                            |
| :--------------------------------------------------------------       |
| O valor de retirado deve esta registrado na lista do caixa            |
